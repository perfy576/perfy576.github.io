Docker安装

ubuntu

Docker 要求 Ubuntu 系统的内核版本高于 3.10 。

输入：wget -qO- https://get.docker.com/ | sh

上面的方式运行的比较慢，可以使用sudo apt install docker.io快速安装。

然后运行sudo service docker start启动服务。
