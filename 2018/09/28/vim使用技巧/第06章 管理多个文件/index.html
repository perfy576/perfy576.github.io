Vim 使用技巧：第06章 管理多个文件

在Vim中，我们所编辑的只是内存中的映像，称为“缓冲区”。

文件存储在磁盘上，而缓冲区只存在于内存中。

管理多个文件

技巧36 用缓冲区列表管理打开的文件

主要是使用vim 1.txt 2.txt，可以同时打开多个文件，界面上只会显示第一个文件。不过后面的2.txt 也是被打开的，被加载到内存中。

相关的命令有；

  • :ls：显示所有打开的命令
  • :buffer 、:b：根据:ls命令显示的文件编号进行，切换到制定编号的文件
  • :bnext；切换到下一个打开的文件，可以循环
  • :bprev：切换到上一个打开的文件，可以循环
  • :bfirst：切换到第一个文件
  • :blast：切换到最后一个文件
  • :bdelete：关闭当前打开的文件，如果带有文件名那么关闭制定的文件，多个文件用空格分隔

如果当前文件被更改，那么切换或是关闭的时候，会有提示，强制切换需要加上!。

同时，如果要q，同时之前的打开的文件都切换到当前过，但是没更改过，那么会关闭所有文件。否则其他的情况都会有提示，需要使用!。

技巧39 将工作区切分成窗口

使用<c-w> s将窗口分成两个水平的窗口；

使用<c-w> v将窗口分成两个竖直的窗口。

然后通过<c-w><c-h/j/k/l>进行窗口的切换。

然后使用:e file将file加载到当前的窗口中。

<c-w> =：所有窗口平铺，等宽等高

<c-w> _：最大化当前窗口的高度，然后其他窗口剩余一行

<c-w> |：最大化当前窗口的宽度，然后其他窗口变为一竖

n <c-w> _：设置高度为n

n <c-w> |：设置宽度为n

技巧40 用标签页将窗口分组

用缓冲区管理窗口，或是使用切换窗口的方式，并不直观。

使用标签页则更能直观的显示当前编辑的是哪个文件。

打开：

  • 在vim中使用命令:tabe file打开文件，即是标签页打开
  • 在bash中，使用-p参数，指定用标签页打开多文件：vim a.txt b.txt -p

当使用切换窗口的方式打开文件的时候，可以使用<c-w> T的方式，将当前窗口移到一个标签页。

切换标签：

  • :tabn：下一个标签，后面带标号，表示第几个
  • :tabp：上一个标签

在普通模式下，使用gt切换到下一个标签，gT切换到上一个标签

关闭标签：

  • :tabc：关闭当前窗口的所有标签
  • :tabo：除了当前标签，关闭其他所有标签
