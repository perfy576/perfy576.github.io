Vim 使用技巧：第07章 打开及保存文件

打开及保存文件

技巧41 用:edit命令打开文件

在vim内打开文件有两种方式：

  • 打开缓冲区，并显示在当前窗口，或是打开在当前窗口：:e file
  • 打开在新标签页：:tabe file

但是打开的时候当前目录是继承自bash的，也就是说，在用上面两个命令打开新的文件的时候的相对路径。

可以使用：%取当前文件路径及其文件名，例如：

输入:e %按tab，就会展开当期那目录后开始到该文件名，而%:h按tab则展开的路径没有本文件名，也就是截取了文件名。

可以使用set autochdir将当前目录设置为当前打开文件所在的目录。

技巧42 用:find打开文件

使用:find命令打开文件，会在一个变量：path指定的文件夹中寻找，文件，也可以使用tab补全。

set path+=app/*

技巧45 以超级用户权限保存文件

当以普通用户权限打开文件，而保存的时候又需要用超级用户权限。

:w !sudo tee %>/dev/nell

原理是w !{cmd}会将输出定向到cmd而不是保存到文件中，然后tee作用是输出的时候同时显示在屏幕，因此，有一个标准输出重定向。
